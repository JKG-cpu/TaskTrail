# Workflow for TaskTrail

## Project Idea

A CLI App (with textual) that can be used to track your school work.

Will be able to 

- Manage classes
    - Create / Remove
    - Calculate GPA / Percentage
- Add assignments to classes
    - Saved in json
- Setup specific study session times
    - i.e a calender to do specific things


# Building Process

## Base

### Logic
1. Start with being able to add / remove classes
    - Be allowed to quickly add assignments to X class
    - Calculations for GPA, Percentage, Letter Grade, etc.
2. Start saving + loading data
3. Start UI

### UI
*Using textual*

1. Screens for viewing classes
    - Adding assignments *(Form)*

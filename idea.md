# Task Trail Layout Ideas

## HomePage
### Contents
    
- Tabs
    - Extra tabs created by profile(s)
    - Settings Tab

- Options to create specific tabs + create tabs
    - To-Dos
    - School tab
        - Able to add classes
        - Keep track of GPA

## Custom Pages
### School Tab

#### Can only have one of these

- Activated by a check in profile creation
    - If school mode = True then create this tab

- Will be able to add different classes
    - Keep track of GPA
    - Keep track of assignments

#### A Reset Option
This will reset the whole entire school tab (get rid of all classes & assignments)

### Todo Tab

#### Can have as many as you want
- Will get a custom name upon creation (default to Todo List) 
- Basic fields
    - Task name
    - Info
    - Completed

#### Option in settings to auto remove ToDo's when complete
